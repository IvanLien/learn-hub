机器学习的三种学习方式

用三个真实案例，直观理解AI如何学习。每个演示都可以亲手互动体验。

① 监督学习 ② 无监督学习 ③ 强化学习
①

监督学习 Supervised Learning

案例：邮件垃圾过滤器 · Email Spam Filter

💡 核心概念

就像老师批改作业：给AI看大量带有正确答案的例子，让它找规律，然后用学到的规律来判断新数据。

每条训练数据都有一个标签（Label），告诉模型"这封邮件是垃圾邮件"或"不是"。

输入
标注好的数据（输入 + 正确答案）
输出
能预测新数据的模型
📚 类比：做题 → 对答案 → 总结规律 → 考试

②

无监督学习 Unsupervised Learning

案例：客户分群 · Customer Segmentation

💡 核心概念

数据里没有任何标签。AI自己探索数据的内部结构，发现隐藏的规律和分组。

电商平台有几百万客户，没人手动给客户贴标签——AI自动发现"高消费年轻人"、"老年节省族"等群体。

输入
无标签的原始数据
输出
自动发现的数据分组/结构
🗂️ 类比：图书管理员把书按相似度归类，没有预设类别

③

强化学习 Reinforcement Learning

案例：机器人走迷宫 · Robot Maze Navigation

💡 核心概念

AI通过试错学习：每次行动后获得奖励或惩罚，逐渐摸索出最优策略。没有标签，没有正确答案——只有结果的反馈。

就像学骑自行车：摔倒（惩罚）→ 调整平衡（策略更新）→ 骑稳了（奖励）→ 越来越好。

输入
环境 + 奖惩信号
输出
最优行动策略（Policy）
🎮 类比：玩游戏 → 失败重来 → 逐渐通关
🤖 = 机器人（从左上出发）

🧱 = 墙（无法通过，-1分）
📍 = 普通移动（-0.1分）
